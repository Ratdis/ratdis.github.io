    之前用的Keepass+Dropbox同步数据库以及Chrome上的记住密码，安卓上同步数据库怪怪的，Chrome同步倒是很棒，但是并没有经常用Keepass的自动填充之类的，而且由于密码并不是放在一块，有些密码只存在keepass或者chrome中还得手动去找出来..

    能忍到现在也是奇迹了吧，不过现在已经整的很舒服了，事后水文就是理直气壮啊

    本文大环境：macOS Mojave 10.14.5 && iOS 12.3.1

    使用KeePass管理密码

    KeePass是一款开源的密码管理工具，没啥说的，下载软件设个主密码就往里面扔平时各种账号密码吧

    macOS上用 KeePassXC

    KeePassXC设置好自动填充快捷键（我的是⌘⌥⇧F）

    自动填充如果没效果参考Link，需要去系统辅助功能里面手动给KeePassXC权限

    iOS上用 FantasyPass

    这里提下使用中遇到的一些小问题

    KeePassXC目前稳定2.4.3会遇到导入CSV文件无法保存数据库文件的问题Link（在导入Chrome密码时遇到的），去这Link下载最新体验版导入后再换回稳定版即可（因为体验版可能不稳定丢失信息，导入数据库后就可以换回了）

    FantasyPass目前如果数据库是WebDAV自动同步的话，在Spotlight中搜索进入回闪退，已经和开发者交流，并表示下个或者下下个版本修复（大概吧

    数据库我是使用坚果云的WebDAV同步，效果不错

    GPG加密

    请直接阅读这篇关于macOS上使用GPG加密的文章Link(英文) 里面有详细的图文操作以及原理解释

    推荐的快捷键设置

    • Enable “OpenPGP: Decrypt” and set its shortcut to ⌃⌥⌘- (i.e., control option command minus)
    • Enable “OpenPGP: Encrypt” and set its shortcut to ⌃⌥⌘= (i.e., control option command equals)
    • Enable “OpenPGP: Sign” and set its shortcut to ⌃⌥⌘[ (i.e., control option command open bracket)
    • Enable “OpenPGP: Verify” and set its shortcut to ⌃⌥⌘] (i.e., control option command close bracket)
